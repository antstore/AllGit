REST API routes that can be opened in the browser:

  • /product List all products
  • /product?limit=3&skip=2 List only 3 products out of them all, and skip first 2 products
  • /product/search/{token} List all products that contains {token} (eg. "dark")
  • /product/{id} Return info on one product, of id {id}
  • /brewery/{id} Prints details about a brewery of id {id} and the detailed list of all its products.
